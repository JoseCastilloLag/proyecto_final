\# Proyecto Final - Modelo Estadístico



\## Descripción

Este proyecto analiza el rendimiento académico de estudiantes utilizando métodos estadísticos y machine learning.



Se aplican:

\- Intervalos de confianza (Bootstrap)

\- Pruebas de hipótesis

\- Modelos de regresión

\- Clasificación

\- Dashboard interactivo



\---



\## Tecnologías utilizadas



\- Python

\- Pandas

\- NumPy

\- Scikit-learn

\- SciPy

\- Plotly

\- Dash



\---



\##  Cómo ejecutar el proyecto



1\. Instalar dependencias:

pip install -r requirements.txt



2\. Generar datos:

python generar\_data.py



3\. Ejecutar modelo:

cd src

python modelo\_completo.py



4\. Ejecutar dashboard:

cd ../dashboard

python app.py



\---



\## Resultados



\- Modelo con buena capacidad predictiva (R² alto)

\- Clasificación precisa (Accuracy alto)

\- Evidencia estadística significativa (p-valor < 0.05)

\- Dashboard interactivo para análisis visual
